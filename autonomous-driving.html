Autonomous Driving Simulation using Reinforcement Learning

This project demonstrates how to train a self-driving car to navigate racetracks using various Reinforcement Learning (RL) algorithms‚ÄîQ-learning, Deep Double Q-Learning (DDQN), and Deep Deterministic Policy Gradient (DDPG)‚Äîwithin a custom-built Pygame simulation.

Algorithms Implemented

  • Q-Learning: Tabular RL with discrete actions and states (based on ray sensors and angle to goal).
  • DDQN: Neural-network based value function with target network and epsilon-greedy exploration.
  • DDPG: Actor-Critic model for continuous control using steering and throttle.

Simulation Environment

  • Built in Pygame with TUM racetracks
  • Vehicle model: Single-track kinematic model
  • Sensors: Simulated LiDAR using ray casting
  • Reward System: Positive reward for forward motion, penalties for collisions, bonus for lap completion

Logging and Visualization

  • TensorBoard: Tracks reward, loss, and performance curves
  • CSV: Logs scores, episode duration, steps, distances
  • Visualization: Path plots of vehicle movement using Matplotlib

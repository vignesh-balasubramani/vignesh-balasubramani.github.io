Autonomous Driving using Reinforcement Learning

A project to train a self-driving race car using Q-learning, DDQN, and DDPG in a Pygame simulation.

Algorithms Used
  • Q-Learning: Discrete state-action mapping using ray sensors and angle to goal.
  • DDQN: Deep Q-learning with target networks and epsilon-greedy exploration.
  • DDPG: Actor-Critic approach for continuous control of throttle and steering.
Simulation Features
  • Custom-built environment using Pygame with TUM racetracks.
  • Vehicle uses single-track kinematic model with LiDAR-style ray sensors.
  • Reward system for staying on track, moving forward, and completing laps.
Training Logs & Visualizations
  • TensorBoard: Reward curves, loss curves, and performance metrics.
  • CSV logs: Per-episode stats like score, distance, and duration.
  • Matplotlib plots: Path visualizations of learned agent behavior.

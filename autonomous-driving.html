Autonomous Driving using Reinforcement Learning
Overview

This project demonstrates how to train a self-driving race car using Q-learning, DDQN, and DDPG within a custom Pygame simulation. The agent learns to drive autonomously using only simulated sensor data and a reward signal.

Algorithms Implemented
  • Q-Learning: Discrete state-action space with tabular learning.
  • DDQN: Value-based deep RL with target networks and epsilon-greedy exploration.
  • DDPG: Actor-critic method enabling continuous control over steering and throttle.
Simulation Environment
  • Custom-built in Pygame simulating a single-track vehicle on TUM racetracks.
  • Ray-based sensors simulate LiDAR for environmental perception.
  • Rewards for forward motion, staying on track, and completing laps; penalties for stalling or collisions.
Logging and Visualization
  • TensorBoard plots for reward, loss, and actor/critic behavior.
  • CSV logs for episode-wise metrics like distance, steps, and cumulative rewards.
  • Matplotlib used for vehicle path visualization and performance plots.
